Preparação do ambiente de desenvolvimento


Muito importante:
  Se você é usuário de Mac ou Linux, infelizmente precisará virtualizar uma máquina Windows, pois o ambiente de desenvolvimento (Visual Studio) com a tecnologia que precisamos é exclusiva para Sistemas Operacionais Windows.

  Neste caso, recomenda-se utilizar a Virtual Box (disponível em https://www.virtualbox.org/) e baixar uma imagem do Windows 10 do próprio site da Microsoft (https://www.microsoft.com/pt-br/software-download/windows10ISO)

  Recomenda-se ter ao menos 50Gb livres e ao menos 8Gb de ram
---------------

Preparação do ambiente

1. Abra a página do Visual Studio: https://visualstudio.microsoft.com/pt-br/
  1.1 Fazer o Download do Visual Studio Comunity
  1.2 Durante o processo de instalação, você precisará selecionar as tecnoloigas que serão instaladas. Para nossa disciplina são essas:
    - ASP.NET e desenvolvimento Web (não obrigatório)
    - Desenvolvimento para desktop com .NET
    - Processamento e armazenamento de dados
  1.3 Clicar para instalar

Após a instalação, fechar o instalador e testar
